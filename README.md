# MentorMan

🌼<h2>프로젝트 소개</h2>
<p>멘토맨 프로젝트는 멘토와 멘티가 활동하는 소셜 네트워크 서비스 플랫폼입니다.
온/오프라인 멘토링 서비스를 통해서 다양한 분야에 진출해 있는 멘토에게 취업, 면접, 학습을 받을 수 있습니다. 
또한, 관심분야가 비슷한 멘티들이 서로 소통을 하며 다양한 정보와 의견을 나눌 수 있는 장을 제공 할 수 있도록 개발하였습니다.</P>
